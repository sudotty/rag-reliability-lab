# Showcase Plan

## Product form

A small console for checking document search quality.

## MVP screens

1. Documents.
2. Chunks.
3. Search comparison.
4. Answer with sources.
5. Metrics.
6. Saved test cases.

## Demo flow

1. Import documents.
2. Inspect chunks.
3. Ask one question.
4. Compare search methods.
5. Show the answer and sources.
6. Save the case for later checks.

## Good demo points

- The user can see the source text.
- The user can compare search methods.
- The user can check whether the answer has evidence.
- The project is practical and easy to explain.
